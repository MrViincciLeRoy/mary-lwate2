# Mary Lwate Website
## Project Overview
This project is a Django-based website for Mama Mary Lwate, showcasing her awards, services, and community work.
## Key Features
* Display of awards and services
* Community work showcase
* Customizable templates
## Tech Stack
* Django
* Python
* HTML/CSS
* JavaScript
## Installation
To install the project, follow these steps:
1. Clone the repository: `git clone https://github.com/username/mary-lwate2.git`
2. Navigate to the project directory: `cd mary-lwate2`
3. Install the required packages: `pip install -r requirements.txt`
4. Run the migrations: `python manage.py migrate`
5. Run the development server: `python manage.py runserver`
## Usage
To use the website, simply navigate to `http://localhost:8000` in your web browser.
## Required Environment Variables
* `DJANGO_SETTINGS_MODULE`: The Django settings module to use.
* `SECRET_KEY`: The secret key for the Django project.
* `DEBUG`: A boolean indicating whether the project is in debug mode.
Note: These environment variables should be set in a `.env` file or in your operating system's environment variables.
